1.7 Describe legal guidelines and constraints that impact digital projects – How does GDPR impact digital projects?

GDPR (General Data Protection Regulation) GDPR impact digital marketing in three ways that's consent, data access and data processing. it will be easier for an individual to withdraw his consent for data processing. So making it clear for people to withdraw their consent at anytime and must be easier for them to do that as it was to give their consent. Digital marketers will only be able to process the limited amount of data they really need, they will be required to have legal justification for why the personal data they've collected is being processed, because of this digital projects will be limited to what can be tried and tested. 1.5 Describing copyright and other constraints affecting websites

Copyright is one of the main types of intellectual property. Intellectual property allows a person to own things they create in the same way as something physical can be owned. It is the right to prevent others copying or reproducing someone's work. Copyright and other intellectual property laws affect websites because it may restrict what you are able to publish. The best way of getting around this is getting permission from the original author to use the content. As a general rule, have the assumption that all works are protected by either copyright or trademark law unless there are clear indications that it isn't. This creates limitations on what can be achieved by using someones great design or an idea.

1.7 Describe which are the appropriate filetypes for websites

It all depends on what sort of website you are working on, I will talk about some of the file types here first one is .CSS — Cascading StyleSheet, CSS files take care of how websites look. CSS files can be created or edited in any text-editor like Notepad. I created and used .css filetype on visual studio during this website project of mine for designing purpose. .DOC — Microsoft Word Document, Microsoft Word is the most popular word processing software in the world. You probably won’t come across loads of doc files, but if you do it can be annoying if you haven’t got a program that can open them. .EXE — EXEcutable file, If you download a program that you need to install, it will likely come as an exe file. .GIF — Graphics Interchange Format, The most common image format on the Internet. Good for simple images. .HTML/ .HTM — HyperText Markup Language file, Most pages you create for a website will be HTML files. I used .html filetype to write the code for the website. .JPG/ .JPEG — Joint Photographic Experts Group file, Another very common image file format, mainly used for photos. Most of the images I used on my website had .jpg format. .PNG — Portable Network Graphics, PNGs are a file format designed to be used in place of GIFs. They are usually slightly smaller, and sport advanced features like alpha-channel transparency and 24-bit colour support. I used .jpg file format for background images, i used .mp4 format for the background video i used for the welcome page. I also used .js file type for javascript.

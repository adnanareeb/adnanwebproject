1.5 Before starting a new project the first thing to consider is to choose the right technology for the project. And in order to do that you will have to look at the requirements for that project like what's needed and how you want your project to turn out. If we suppose we want to build a website then we have to first think of how we want our website to look and whats the content structure. So we will have to see if we need to install any tools to help us with our project. You will have to consider the server side of it, what's best suited for the type of website you are trying to build, if its a massive project with a heavy design and load then its worth looking at some of the faster servers so it won't make your website slow. You will have to decide if you want to stick with javascript or use server side rendering using MVC or an all-in-one CMS. CMS will make sure your website is up and running in no time. Deciding what sort of template you want to use with your website.
